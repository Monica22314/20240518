以下是堆積排序的基本步驟：

建立最大堆積（Max Heap）或最小堆積（Min Heap）：
如果要遞增排序，則建立最大堆積，其中父節點的值大於子節點。
如果要遞減排序，則建立最小堆積，其中父節點的值小於子節點。
排序過程：
將根節點（最大值或最小值）與最後一個節點交換。
移除最後一個節點（已排序的部分）。
重新調整堆積，使其滿足堆積性質。
重複以上步驟，直到所有節點都被排序。
堆積排序的時間複雜度為O(nlogn)，其中n是要排序的元素數量。1234
